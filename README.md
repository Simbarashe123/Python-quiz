# Python-quiz

Projektbeschreibung – Python-Quizprogramm

Dieses Projekt ist ein interaktives Quizprogramm, das mit Python entwickelt wurde.
Das Programm fragt den Namen des Nutzers ab und stellt fünf Python-bezogene Fragen.
Die Antworten werden überprüft, die Punktzahl wird berechnet und am Ende kann der Nutzer entscheiden, ob er das Quiz erneut spielen möchte.

Verwendete Technologien und Konzepte

Python
Funktionen
Schleifen
Listen
Bedingungen (if/else)
